# 社畜反击战

纯 HTML + CSS + JavaScript Canvas 竖屏全屏 Boss Rush 小游戏。页面按手机小游戏全屏体验设计，双击 `index.html` 或在手机浏览器打开即可游玩。

## 文件结构

```text
社畜反击战/
├── index.html
├── README.md
└── js/
    ├── data.js
    ├── input.js
    ├── entities.js
    ├── game.js
    └── main.js
```

## 触屏操作

底部操作区左右均分：左侧负责移动，右侧负责攻击和系统操作。

- `← / →`：移动
- `跳`：跳跃
- `蹲`：下蹲
- `普攻`：键盘连击
- `反弹`：需求反弹
- `突刺`：离职信突刺
- `远程`：扔订书机
- `大招`：已读不回
- `开始`：开始 / 暂停 / 通关后重开

键盘仍可作为桌面调试备选：方向键移动，`Space` 跳跃，`S` 下蹲，`1/2/3 + J` 出招，`K` 远程，`L` 大招，`Enter` 开始或暂停。

## 招式

触屏按钮会直接释放对应招式：

- `普攻 / 键盘连击`：稳定近战三连击，适合清小怪和贴身打 Boss。
- `反弹 / 需求反弹`：短暂架起蓝色反弹盾，把 Boss 的飞行道具反打回去，对 PPT 光波、大饼、裁员风暴这类招式特别有用。
- `突刺 / 离职信突刺`：向前爆发突进并打出高伤害，适合 Boss 放完硬直后的反击。

每种招式都有独立特效和浮字反馈：键盘连击是黄色/粉色连击斩，需求反弹是蓝色盾面，离职信突刺会打出辞职信冲刺轨迹。

## 玩法

你是一名忍无可忍的普通打工人，要从开放工位区一路打到顶楼 CEO 室。

每一层都有跑图段和 Boss 战：

1. 1F 开放工位区：组长·卷王
2. 2F 会议室：主管·会霸
3. 3F 经理办公室：经理·甩锅侠
4. 4F 总监楼层：总监·画饼大师
5. 5F 顶楼 CEO 室：CEO·资本之神

玩家有体力条和心态条。体力归零会倒下，心态被打空会进入“破防”状态并短暂无法操作。主角头顶会常驻显示当前体力/体力上限，方便战斗中直接判断还能扛几下。拾取咖啡能回血和加速，拾取订书机补充远程弹药，每层隐藏一个带薪拉屎点，碰到后会回满心态。

原地不动 3 秒会进入摸鱼状态并缓慢恢复心态。摸鱼次数、被打倒次数、通关时间会影响结局称号。

当前版本偏向爽快闯关：主角体力和心态上限更高，受击后有更长无敌时间，咖啡回复更多，敌人与 Boss 的攻击频率也更友好。

## 调整建议

游戏参数集中在 `js/data.js`。可以在那里调整 Boss 血量、场景颜色、敌人数量、补给位置和障碍位置。
